# COMANDOS PARA COPIAR E COLAR NO VPS

## ✅ BOT JÁ ESTÁ RODANDO!

## Ver logs em tempo real (para testar /tiktok)
pm2 logs affiliate-bot --lines 50

## Parar de ver logs
Ctrl + C

## Ver status do bot
pm2 list

## Reiniciar bot (se precisar)
pm2 restart affiliate-bot

## Parar bot
pm2 stop affiliate-bot

## Ver últimas 100 linhas de log
pm2 logs affiliate-bot --lines 100

---

## 🎵 TESTAR COMANDO /TIKTOK

1. Abra o terminal e rode:
   pm2 logs affiliate-bot --lines 50

2. No Telegram, envie uma FOTO com a legenda:
   /tiktok https://www.tiktok.com/@loja/video/123 Controle_PS4 89.90 informatica

3. Veja os logs no terminal para verificar se funcionou

4. Abra o site para ver se o produto apareceu
